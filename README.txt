Description du site :

  Le site est une plateforme de réservation de services permettant aux utilisateurs de rechercher différents professionnels dans plusieurs domaines et de réserver des rendez-vous ou des sessions avec eux.

Instructions d’utilisation :

          Ouvrir le fichier index.html
          Cliquer sur “Explore Services” pour accéder aux catégories
          Choisir une catégorie pour afficher les professionnels correspondants
          Cliquer sur “View Details” pour consulter les informations complètes d’un professionnel
          Cliquer sur “Book Now” pour accéder au formulaire de réservation
          Remplir le formulaire puis confirmer le rendez-vous
          Créer un compte ou se connecter depuis la page d’authentification
